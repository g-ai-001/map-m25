# map-m25

仿高德地图 Android 离线单机应用

## 功能特性

- 地图显示与交互（缩放、拖拽、旋转）
- 地图图层切换（普通、卫星、地形）
- 指南针与比例尺叠加层
- 距离测量工具
- 位置分享功能
- 位置定位
- 地点搜索
- 收藏夹管理
- 路线规划（驾车、步行、骑行）
- 设置功能（深色模式、默认缩放、屏幕常亮、高刷新率）

## 技术栈

- Kotlin 2.3.20
- Jetpack Compose + Material 3
- MVVM + Clean Architecture
- Hilt 依赖注入
- Room 数据库
- DataStore 键值存储
- Coil 图片加载
- Navigation Compose
