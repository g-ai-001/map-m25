# map-m25

仿高德地图 Android 离线单机应用

## 功能特性

- 地图显示与交互
- 位置定位
- 地点搜索
- 收藏夹管理
- 路线规划（驾车、步行、骑行）
- 设置功能

## 技术栈

- Kotlin 2.3.20
- Jetpack Compose + Material 3
- MVVM + Clean Architecture
- Hilt 依赖注入
- Room 数据库
- DataStore 键值存储
- Coil 图片加载
- Navigation Compose
