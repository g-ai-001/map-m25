# map-m25

仿高德地图 Android 离线单机应用

## 功能特性

- 地图显示与交互（缩放、拖拽、旋转、3D视角）
- 地图图层切换（普通、卫星、地形）
- 指南针与比例尺叠加层
- 距离测量工具
- 位置分享功能
- 位置定位
- 地点搜索
- 收藏夹管理
- 路线规划（驾车、步行、骑行）
- 地图标记功能（添加、编辑、删除地图标记）
- 足迹记录功能（记录用户轨迹）
- 轨迹回放功能（播放控制、倍速播放）
- 坐标转换工具（度分秒↔十进制）
- 离线地图区域管理
- 热点POI推荐
- 地图快照查看
- 深色主题支持
- 个性化地图样式（5种配色方案）
- 语音播报功能（导航播报、位置提醒）
- 地图注记显示控制（POI标签、道路名称、交通标识、建筑物标注）
- 标注导入功能（GPX/KML格式）
- 设置功能（深色模式、默认缩放、屏幕常亮、高刷新率）

## 技术栈

- Kotlin 2.3.20
- Jetpack Compose + Material 3
- MVVM + Clean Architecture
- Hilt 依赖注入
- Room 数据库
- DataStore 键值存储
- Coil 图片加载
- Navigation Compose
