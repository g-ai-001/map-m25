# 项目规划

## 项目概述
- **项目名称**: map-m25
- **包名**: app.map_m25
- **类型**: 仿高德地图 Android 离线单机应用
- **目标SDK**: Android 16 (API 36)
- **最低SDK**: Android 16 (API 36)

## 技术栈
- JDK 25
- Gradle 9.4.1
- Kotlin 2.3.20
- Jetpack Compose + Material 3
- MVVM + Clean Architecture
- Hilt 依赖注入
- Room 数据库
- DataStore 键值存储
- Coil 图片加载
- Navigation Compose
